# WeatherDashboard
## Purpose
Use 3rd party API to display current weather and also a 5-day forecast. Also create a clickable search history

### The current weather should display
* Input City Name
* Date
* Weather Icon
* Temperature
* Humidity
* Wind Speed
* UV Index

### 5-day Forecast should display
* Date
* Temperature
* Humidity
* Forecast Icon

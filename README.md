# WeatherDashboard
## Purpose
Create a Weather Dashboard that will provide the current weather including
* Input City Name
* Date
* Weather Icon
* Temperature
* Humidity
* Wind Speed
* UV Index
5-day Forecast including
* Date
* Temperature
* Humidity
* Forecast Icon
